# Documentation


## Create project
```bash
$ ng new mystore
$ cd mystore
$ npm i @ngrx/store @ngrx/effects @ngrx/store-devtools @ngrx/schematics --save
$ ng config cli.defaultCollection @ngrx/schematics
$ ng generate store AppState --root --module app.module.ts
```

## Create actions
```bash
$ ng generate action actions
```
